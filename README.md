# Ana_Puscasu_Emma_Stan
Proiectul urmărește să analizeze factorii care influențează aprecierea și popularitatea cărților folosind un dataset de aproximativ 3000 de titluri colectate automat din API-ul OpenLibrary. Scopul principal este de a prezice ratingul mediu al unei cărți (calitatea percepută) și numărul de rating-uri (popularitatea) printr-un proces de analiză predictivă care combină date numerice și textuale.
